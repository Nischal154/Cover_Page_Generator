# Cover_Page_Generator
smart Assignment Cover Page Generator built using Node.js, Express.js, EJS, Bootstrap, and custom CSS.


This project is a simple and smart Assignment Cover Page Generator built using Node.js, Express.js, EJS, Bootstrap, and custom CSS.
It allows students to enter their details in a form and automatically generates a clean, formatted A4-size cover page, ready for download or print.
